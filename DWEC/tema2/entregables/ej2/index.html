Ejercicio 2

Crea una página web con un script que contenga un objeto de la clase persona, la cual tendrá los siguientes atributos:

  • Nombre: Dimas
  • Edad: 28
  • Altura: 185

Una vez creado el objeto, se deberá mostrar por consola: “El usuario Dimas tiene 28 años y mide 185 centímetros”. Todo ello utilizando los atributos del objeto.
