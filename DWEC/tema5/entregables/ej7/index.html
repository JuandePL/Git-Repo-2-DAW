¿Qué es la API Fetch? Indica si puede ser una alternativa a las formas de comunicación asíncrona vistas hasta el momento.

API Fetch es una interfaz que permite al navegador hacer peticiones HTTP a servidores web sin necesidad de AJAX o XMLHttpRequest.

Funciona igual que el ulimo objeto mencionado, pero con una sintaxis mucho más sencilla.
